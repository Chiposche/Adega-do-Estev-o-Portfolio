## 📖 O Desafio
Gerenciar uma coleção de cachaças premium exige muito mais do que apenas uma planilha estática. Colecionadores enfrentam a dificuldade de apresentar rótulos exclusivos com a sofisticação que eles merecem, mantendo o catálogo acessível, atualizado e organizado. O desafio era criar uma plataforma que fosse ao mesmo tempo uma ferramenta de gestão poderosa e uma vitrine de luxo.
## 💡 A Solução
Desenvolvi a **Adega do Estevão**, uma aplicação web *full-stack* completa, focada na experiência do usuário e em um design premium. A plataforma permite que o administrador tenha controle total do seu estoque, enquanto oferece uma vitrine impecável e instalável em qualquer dispositivo móvel.
## ✨ Principais Funcionalidades
* 🤖 **Inteligência Artificial (Remoção de Fundo):** Upload automático de imagens das garrafas com remoção de fundo nativa via IA, garantindo que as fotos fiquem uniformes e com aspecto de estúdio, independente de onde foram tiradas.
* 📱 **PWA (Progressive Web App):** A aplicação pode ser instalada diretamente no smartphone (iOS e Android), entregando uma experiência imersiva e nativa.
* 🥃 **Catálogo Inteligente:** Listagem filtrável por tipo de barril (Amendoim, Carvalho, Jequitibá, etc.), idades e origens.
* ⚙️ **Painel Administrativo:** Área restrita para cadastro, edição e gestão intuitiva de toda a coleção e controle de estoque.
* 🎨 **Design Luxury:** Identidade visual premium, responsiva e desenvolvida para funcionar com maestria tanto no desktop quanto em telas mobile.
 ## 🛠️ Tecnologias Utilizadas
 **Front-end:** React.js, Context API, CSS (Design Responsivo e UI Premium).
 **Back-end & Banco de Dados:** Supabase (Autenticação, Banco de Dados Relacional PostgreSQL e Storage).
 ## 🚀 Impacto & Aprendizados
Este projeto foi um excelente laboratório para integrar Inteligência Artificial ao fluxo de trabalho de usuários comuns, simplificando tarefas trabalhosas (como editar fotos de produtos). Além disso, consolidou meus conhecimentos em Progressive Web Apps (PWA) e na criação de animações programáticas em vídeo usando Remotion para a campanha de lançamento.
